## About This Repo

This is a collection of example extensions for [fp](https://fp.dev), Fiberplane's agent-native issue tracker. The goal is to provide three polished, well-documented extensions that demonstrate different capabilities of the fp extensions API — useful both as learning resources and as starting points for building your own.

Extensions live in `.fp/extensions/` as TypeScript files and hook into fp's issue lifecycle (creation, status changes, comments, etc.).
